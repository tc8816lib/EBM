# TCEBM 專案開發與優化計畫

## 專案概述
本專案為 TCEBM 課程落地頁面，旨在提供學員查看課程安排、AI Prompt、資源連結與競賽資訊。本專案採用 **Frontend (HTML/JS) + GAS Backend (API)** 的架構，並持續優化行動端的使用者體驗。

## 目前狀態 (Current Progress)

### 1. UI/UX 與 結構優化 (已完成)
- [x] **高密度行動介面**：優化表格間距，類別標籤與按鈕整合至單列。
- [x] **實體/線上課程區分**：透過背景色 (`.tr-physical`, `.tr-online`) 與標籤強化視覺區隔。
- [x] **課程分桶管理**：實作「今日課程」、「未來課程」與「已結束課程」的分群顯示。
- [x] **過往課程年度分組**：將已結束課程按年份分組 (Accordion)，並預設僅展開當年度，提升頁面整潔度。
- [x] **動態類別標籤雲**：實作課程類別篩選功能，支援 AI、行銷、職能等多維度過濾。
- [x] **資源連結優化**：移除冗餘時間欄位，整合 🔗 圖示，提供更直覺的跳轉體驗。
- [x] **最新訊息橫幅**：頂部固定顯示最新公告，支援預約發佈與 URL 自動轉連結。

### 2. 性能與功能開發 (已完成)
- [x] **全量資料預載 (Cache-First)**：實作 `getInitialData` 一次性抓取所有 Sheet 資料，大幅減少載入次數。
- [x] **背景同步與快取**：整合 LocalStorage 機制，達成「秒開」體驗並在背景靜默更新資料。
- [x] **Google 一鍵登入**：完整整合 GSI 驗證與自動登入 (One Tap) 功能。
- [x] **權限控制系統**：根據登入 Email 動態顯示「管理員功能」與「確認出席」按鈕。
- [x] **Prompt 複製功能**：提供一鍵複製 Prompt 文字，優化行動端操作。

## 後續執行任務 (Next Steps)

### 3. 進階功能與體驗優化
- [ ] **PWA 支援強化**：配置 `manifest.json` 與 Service Worker，強化離線瀏覽能力與桌面/手機捷徑圖示。
- [ ] **黑暗模式精進**：優化 `[data-theme="dark"]` 的色彩對比度，特別是類別標籤與表格文字的易讀性。
- [ ] **管理者後端功能**：實作 Admin 浮動按鈕對應的資料新增 Modal，並串接 GAS 寫入 API。
- [ ] **課程影音深度整合**：優化課程回放連結的呈現方式，可能加入 YouTube 預覽或區分不同平台的圖示。
- [ ] **出席統計導出**：為管理員增加一鍵導出出席名單（或查看簡易統計）的功能。

## 驗證計畫
1. **API 連線測試**：確認各分頁資料能透過 `fetch()` 從 GAS 正常抓取並正確快取。
2. **跨平台相容性**：在 iOS/Android 不同瀏覽器下測試課程年份收合與類別標籤的點擊行為。
3. **離線情境測試**：關閉網路後確認是否能正常顯示 LocalStorage 中的最後一次快取資料。
